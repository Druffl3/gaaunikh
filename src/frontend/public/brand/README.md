Brand assets for the static storefront belong here.

Drop these files in the repo:

- `src/frontend/public/brand/hero-home.jpg` or `hero-home.png`
- `src/frontend/public/brand/logo.png`
- `src/frontend/app/icon.png`

Notes:

- Prefer `hero-home.jpg` for the hero background unless the image needs transparency.
- Use `logo.png` for the brand logo in the header.
- Use a square `icon.png` for the favicon. A 512x512 source is a good default.
